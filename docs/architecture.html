Prehľad systému

Kompletná automatizácia predajného procesu FV systémov — od prvého kontaktu po záväznú ponuku — bez ľudského obchodníka. Zákazník interaguje výhradne s AI botom, ktorý zbiera dáta, počíta optimálny systém a generuje personalizovanú ponuku.

Sales Flow — 8 krokov

Celý proces od prvého kontaktu po podpísanú zmluvu prebieha automaticky. Ľudský obchodník zasahuje iba pri väčších firemných projektoch (nad 20 kWp).

1

Lead zachytený v Odoo

Zákazník vyplní formulár na webe, klikne na reklamu alebo zavolá. Kontakt sa automaticky vytvorí v Odoo CRM.

⏱ 0 sekúnd
2

Odoo spustí n8n webhook

Odoo automation rule pošle POST request na n8n webhook s dátami kontaktu (meno, email, telefón, zdroj).

⏱ automaticky pri uložení
3

n8n zavolá AI Bot API

n8n pošle lead dáta na /webhook/odoo/new-lead. Bot vytvorí novú konverzáciu a vráti conversation_id + prvú správu.

⏱ <2 sekundy
4

Email s personalizovaným odkazom

n8n pošle zákazníkovi email s unikátnym odkazom na chat widget. URL obsahuje conversation_id pre bezproblémové pokračovanie.

⏱ do 30 sekúnd od záchytu leadu
5

AI konverzácia (7 krokov)

Claude AI zbiera: typ nehnuteľnosti → spotreba → strecha → lokalita → budget → termín → kontakt. Každá otázka je jedna správa — prirodzený dialóg.

⏱ ~5 minút zákazníkovho času
6

Generovanie personalizovanej ponuky

PV Calculator vypočíta optimálny výkon (kWp), cenu, ročnú úsporu a návratnosť investície. Claude AI to prezentuje v čitateľnej forme.

⏱ okamžite po zozbieraní dát
7

Aktualizácia Odoo + notifikácia tímu

n8n aktualizuje lead v Odoo: stage = Qualified, doplní zozbierané dáta. Obchodný tím dostane notifikáciu s kompletným súhrnom.

⏱ automaticky po dokončení chatu
8

Follow-up pre nekvalifikovaných

Zákazníci, ktorí chat nedokončili, dostanú follow-up email po 3 dňoch. Celý cyklus sa opakuje.

⏱ po 3 dňoch (n8n Schedule)

Komponenty

🤖 AI Bot API

Jadro systému. Spravuje konverzácie, volá Claude API, spracúva tool calls (update_lead_data, generate_proposal), uchováva session state.

FastAPI · Python 3.11

🧠 Claude AI Engine

claude-haiku-4-5 model s PV expert systémovým promptom. Používa tool use pre štruktúrované zber dát a spustenie kalkulácie.

Anthropic API · Tool Use

⚡ PV Calculator

Výpočtová logika: kWp z spotreby, slnečná mapa SR, ROI kalkulácia, odporúčanie batérie. Pure Python, plne testovateľné.

Python · Bez externých závislostí

🔄 n8n Orchestrator

Orchestruje celý workflow: Odoo webhook → AI Bot → email → wait → check → update Odoo → notifikácia.

n8n Cloud · 10 nodes

💬 Chat Widget

Embedovateľný vanilla JS widget bez závislostí. Funguje na akomkoľvek webe. Podporuje typing indicator, proposal card, quick reply.

HTML/CSS/JS · Zero dependencies

🏢 Odoo CRM

Existujúci CRM systém. Integrovaný cez webhook (outbound) a XML-RPC API (inbound update). ECO PRODUKT nemusí meniť procesy.

Odoo · XML-RPC API
