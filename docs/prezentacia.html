Analýza zadania

Čo zadanie hovorí a čo som z toho pochopil

Zadanie hovorí

"Lead s kontaktom bude importovaný do Odoo. Zvyšok architektúry navrhnite Vy."

Moje pochopenie

Odoo je vstupný bod — nesmiem ho obísť. Celá automatizácia začína v Odoo a končí aktualizáciou v Odoo.

Preto som navrhol: Odoo Automation Rule → webhook → n8n → AI bot → zákazník → n8n → Odoo update. Odoo ostáva master systém pre všetky dáta.

Moje uvažovanie

Prečo vlastný AI bot a nie len n8n?

1

n8n-only bot má limity v konverzácii

Predaj FV systémov vyžaduje prirodzený dialóg — zákazník píše "platím okolo 80 eur mesačne", nie "spotreba: 5040 kWh/rok". n8n nevie interpretovať voľný text.

2

Claude AI rieši neštruktúrovaný vstup

LLM preloží "malý domček, platíme málo" na konkrétne parametre pre kalkuláciu. Zároveň kladie kontextové otázky — nie robotický formulár.

3

n8n ostáva — robí čo robí dobre

Orchestrácia (kedy, čo, komu), emailové notifikácie, Odoo update, follow-up plány. AI bot = inteligencia. n8n = plán B a C.

4

Tool use pre štruktúrované dáta

Bot používa Anthropic "tool calling" — keď zákazník poskytne dostatok dát, automaticky zavolá PV Calculator a vygeneruje ponuku.
